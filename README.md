Introdução 
========
> Seguem abaixo informações de como fazer requisições no backend da cleanpool.  
Ex: 
```sh
http://localhost:3000/usuario/cadastrar
```
ou
```sh
http://UrlDoServidor:3000/usuario/cadastrar
```
> Obs: Todas as requisições precisam de token.
